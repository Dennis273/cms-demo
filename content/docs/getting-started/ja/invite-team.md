# チームメンバーの招待

同僚を企業メールチームに招待する方法を学びましょう。

## 招待方法

### 個別招待

1. 管理コンソールに移動
2. 「ユーザー管理」>「メンバーを招待」をクリック
3. メンバーのメールアドレスを入力
4. ユーザーロールを選択
5. 「招待を送信」をクリック

### 一括招待

CSVファイルでメンバーを一括招待：

1. CSVテンプレートをダウンロード
2. メンバー情報を記入
3. CSVファイルをアップロード
4. 確認して招待を送信

## ユーザーロール

- **管理者**：すべての機能にフルアクセス
- **メンバー**：基本的なメール機能
- **読み取り専用**：閲覧のみのアクセス

## 招待の有効期限

招待リンクの有効期限は7日間です。期限切れ後は再度招待を送信する必要があります。
