# アカウントの作成

ニャンメールへようこそ！このガイドでは、アカウントの作成と使用開始方法をご説明します。

## 登録手順

1. [登録ページ](/signup)にアクセス
2. 企業メールアドレスを入力
3. 安全なパスワードを設定
4. メールアドレスを確認
5. アカウント設定を完了

## パスワード要件

アカウントのセキュリティを確保するため、パスワードは以下の要件を満たす必要があります：

- 8文字以上
- 大文字と小文字を含む
- 少なくとも1つの数字を含む
- 少なくとも1つの特殊文字を含む

## メール確認

登録後、確認リンクが記載されたメールをお送りします。24時間以内にリンクをクリックして確認を完了してください。

確認メールが届かない場合は、迷惑メールフォルダを確認するか、「確認メールを再送信」をクリックしてください。
