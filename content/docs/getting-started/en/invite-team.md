# Invite Team Members

Learn how to invite colleagues to join your enterprise email team.

## Invitation Methods

### Single Invitation

1. Go to the admin console
2. Click "User Management" > "Invite Members"
3. Enter member email address
4. Select user role
5. Click "Send Invitation"

### Bulk Invitation

Bulk invite members via CSV file:

1. Download CSV template
2. Fill in member information
3. Upload CSV file
4. Confirm and send invitations

## User Roles

- **Administrator**: Full access to all features
- **Member**: Basic email functionality
- **Read-only**: View-only access

## Invitation Validity

Invitation links are valid for 7 days. After expiration, a new invitation must be sent.
