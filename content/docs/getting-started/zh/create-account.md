# 创建账户

欢迎使用喵喵企业邮箱！本指南将帮助您快速创建账户并开始使用。

## 注册步骤

1. 访问 [注册页面](/signup)
2. 输入您的企业邮箱地址
3. 设置安全密码
4. 验证邮箱地址
5. 完成账户设置

## 密码要求

为了保障账户安全，密码需满足以下要求：

- 至少 8 个字符
- 包含大小写字母
- 包含至少一个数字
- 包含至少一个特殊字符

## 验证邮箱

注册后，我们会向您的邮箱发送验证链接。请在 24 小时内点击链接完成验证。

如果没有收到验证邮件，请检查垃圾邮件文件夹，或点击"重新发送验证邮件"。

## 下一步

账户创建完成后，您可以：

- [配置企业域名](/docs/getting-started/configure-domain)
- [邀请团队成员](/docs/getting-started/invite-team)
- 开始使用邮箱服务
