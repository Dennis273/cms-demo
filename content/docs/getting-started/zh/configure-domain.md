# 配置域名

将您的企业域名与喵喵邮箱绑定，让您的团队使用专业的企业邮箱地址。

## 添加域名

1. 登录管理控制台
2. 进入"域名管理"设置
3. 点击"添加域名"
4. 输入您的域名（如：example.com）

## DNS 配置

添加域名后，您需要配置以下 DNS 记录：

### MX 记录

| 主机 | 类型 | 优先级 | 值 |
|------|------|--------|-----|
| @ | MX | 10 | mx1.meowmail.com |
| @ | MX | 20 | mx2.meowmail.com |

### SPF 记录

```
v=spf1 include:spf.meowmail.com ~all
```

### DKIM 记录

DKIM 记录将在域名验证后自动生成。

## 验证域名

完成 DNS 配置后，点击"验证域名"按钮。验证可能需要几分钟到 48 小时不等。

## 常见问题

**Q: DNS 记录多久生效？**

A: DNS 记录通常在几分钟内生效，但完全传播可能需要 24-48 小时。

**Q: 可以添加多个域名吗？**

A: 是的，专业版及以上可以添加多个域名。
