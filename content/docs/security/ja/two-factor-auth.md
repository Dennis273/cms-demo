# 二要素認証

二要素認証（2FA）を有効にして、アカウントにセキュリティレイヤーを追加しましょう。

## 二要素認証とは

二要素認証は、ログイン時に2種類の確認を要求します：
1. パスワード（知っているもの）
2. 動的コード（持っているもの）

## 設定手順

1. 設定 > セキュリティに移動
2. 「二要素認証を有効にする」をクリック
3. 認証アプリでQRコードをスキャン
4. 6桁のコードを入力して確認

## 対応認証アプリ

- Google Authenticator
- Microsoft Authenticator
- Authy
- 1Password

## バックアップコード

認証デバイスを紛失した場合に備えて、有効化後にバックアップコードを保存してください。
