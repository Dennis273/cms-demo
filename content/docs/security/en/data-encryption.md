# Data Encryption

Learn how MeowMail protects your data.

## Encryption Methods

### Transport Encryption

All data transfers use TLS 1.3 encryption for secure transmission.

### Storage Encryption

Email data is stored with AES-256 encryption.

### End-to-End Encryption

Professional and higher plans support end-to-end encryption - only sender and recipient can read email content.

## Security Certifications

- SOC 2 Type II
- ISO 27001
- GDPR Compliant
