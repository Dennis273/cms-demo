# Two-Factor Authentication

Enable two-factor authentication (2FA) to add an extra layer of security to your account.

## What is Two-Factor Authentication

Two-factor authentication requires you to provide two forms of verification when logging in:
1. Password (something you know)
2. Dynamic code (something you have)

## Setup Steps

1. Go to Settings > Security
2. Click "Enable Two-Factor Authentication"
3. Scan the QR code with an authenticator app
4. Enter the 6-digit code to confirm

## Supported Authenticators

- Google Authenticator
- Microsoft Authenticator
- Authy
- 1Password

## Backup Codes

Save your backup codes after enabling, in case you lose access to your authenticator device.
