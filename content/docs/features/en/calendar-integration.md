# Calendar Integration

Seamlessly integrate email with calendar to improve schedule management.

## Features

- Auto-detect meeting invitations in emails
- One-click add to calendar
- Meeting reminder notifications
- Sync with Google Calendar, Outlook

## How to Use

1. Go to Settings > Calendar Integration
2. Connect your calendar account
3. Enable auto-sync

## Supported Calendars

- Google Calendar
- Microsoft Outlook
- Apple iCloud
- CalDAV compatible calendars
